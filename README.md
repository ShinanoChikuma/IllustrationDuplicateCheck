# IllustrationDuplicateCheck
## 简介
一种基于**ResNet18**和**余弦相似度**的图片查重
## 功能
可将同一文件夹路径下的所有图片进行查重，将重复的图片两两移动到子一级的文件夹里，由操作者自行选择保留或删除
## 使用方法
- 运行后输入**文件夹路径**，回车
## 运行逻辑
- 遍历该文件夹里所有图片（`.jpg`， `.jpeg`， `.png`， `.bmp`）的路径
- 自动检测是否有可用的 GPU，并优先使用 GPU 加速计算
- **加载ResNet-18提取图像特征**
  - 调整图片大小为`224x224`
  - 转换为PyTorch张量
  - 归一化到ImageNet训练时的均值与标准差
  - 打开图片并转换为RGB
  - 进行预处理
  - 送入ResNet-18提取特征
  - 返回NumPy数组格式的特征向量
- **计算相似度**
  - 遍历所有图片，提取特征并存入字典
  - 转换为NumPy数组
  - 计算余弦相似度矩阵
- **查找重复图片**
  - 遍历余弦相似度矩阵，找到相似度大于`0.98`的图片对
- **处理重复图片**
  - 创建重复图片文件夹
  - 为每组重复图片创建子文件夹
  - 移动重复图片到对应的子文件夹
## 不足之处
- 在处理差分图（指图片大体结构不变，但表情等有细微改变的图）时，会误判为同一张图
- 运行速度较慢
- 内存占用较大
